# Food-Nutrition-MVVM
Build food nutrition app with Clean Architecture and using Spoonacular API


Tech are used:
- Dependency Injection with Hilt
- Android Navigation and SafeArgs
- MVVM
- DataStore
- Room
- Online offline using DataSource
- Coroutines
- Shimmer Loading
